Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje utilizado para describir la presentación de un documento HTML. Permite definir el diseño, el formato y el aspecto visual de una página web.

Con CSS, puedes controlar aspectos como el color del texto, el tamaño de las fuentes, la disposición de los elementos en la página, el espaciado, los márgenes y mucho más.

Selección de elementos

Con CSS, puedes seleccionar elementos HTML utilizando diferentes tipos de selectores, como selectores de etiquetas, de clase, de ID y de atributo.

Por ejemplo:

                    
/* Selector de etiqueta */
p {
    color: blue;
}

/* Selector de clase */
.highlight {
    background-color: #f8f9fa;
}

/* Selector de ID */
#header {
    font-size: 24px;
}

/* Selector de atributo */
input[type="text"] {
    border: 1px solid #ccc;
}

Estos son solo algunos ejemplos básicos de lo que se puede lograr con CSS. A medida que aprendas más sobre este lenguaje, podrás crear diseños web más complejos y atractivos.
