Introducción a HTML

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Consiste en una serie de elementos llamados "etiquetas" que estructuran el contenido de una página web.

Cada etiqueta HTML describe diferentes partes del contenido de la página y su función. Por ejemplo, la etiqueta <h1> se utiliza para crear encabezados, mientras que la etiqueta <p> se utiliza para párrafos de texto.

Además de las etiquetas de contenido, HTML también utiliza etiquetas para definir la estructura y el diseño de la página. Estas etiquetas incluyen elementos como <div>, <span>, y <img>.

Una de las características más importantes de HTML es su capacidad para crear enlaces utilizando la etiqueta <a>. Esto permite que las páginas web estén interconectadas, lo que facilita la navegación entre diferentes páginas y sitios web.
